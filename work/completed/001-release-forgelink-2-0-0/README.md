## Intent

Record the first ForgeLink release as durable history. This release renamed the
app, modernized the Electron/React/runtime surface, preserved local data
recovery, and published the first GitHub installer release.

## Decisions

The product name is ForgeLink because the scope is broader than voice and
messaging: direct human communication, trusted signals, and future agentic app
channels.

## Scope

README, local tests, build artifacts, and GitHub release `v2.0.0`.

assets, and the validation commands captured in evidence.
